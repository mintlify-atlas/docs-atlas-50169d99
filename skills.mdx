# Skills & Technologies

As a Multiplatform Application Developer student at IES Oretania, I'm building expertise across a broad range of technologies, platforms, and development practices.

## Multiplatform Development

Multiplatform application development involves creating software that runs across different operating systems and devices from a shared codebase, while maintaining native performance and user experience.

  </Card>
